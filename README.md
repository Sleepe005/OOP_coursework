# OOP_coursework
На перекрестке двух равноценных дорог установлены светофоры.
Светофоры управляются с помощью пульта управления. Пульт может быть выключен или включен
 - При выключенном пульте светофоры не горят. 
 - При включенном пульте он может находиться в состояниях «автономное» (2), «управление ручное» (3) и «управление автоматическое» (1).
   - (2) В состоянии «автономное» все светофоры мигают желтым
   - (1) В состоянии «управление автоматическое» светофоры по заданному регламенту времени меняют свое состояние «зеленое» - «желтое» - «красное» «желтое» - «зеленое».
   - (3) В состоянии «управление ручное» все согласованные состояния светофоров задаются с пульта вручную без соблюдения временного регламента.

## Протокол обмена данными пульт ---> светофор

    ### режим работы 1
    [время красного сигнала] [время жёлтого сигнала] [время зелёного сигнала]
    ### режим работы 3
    [номер светофора] [цвет сигнала]
    
 - [номер светофора] -->(ответ) [горящий сигнал]
