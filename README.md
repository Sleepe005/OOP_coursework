# OOP_coursework
На перекрестке двух равноценных дорог установлены светофоры.
Светофоры управляются с помощью пульта управления. Пульт может быть выключен или включен
 - При выключенном пульте светофоры не горят. 
 - При включенном пульте он может находиться в состояниях «автономное», «управление ручное» и «управление автоматическое».
   - В состоянии «автономное» все светофоры мигают желтым
   - В состоянии «управление автоматическое» светофоры по заданному регламенту времени меняют свое состояние «зеленое» - «желтое» - «красное» «желтое» - «зеленое».
   - В состоянии «управление ручное» все согласованные состояния светофоров задаются с пульта вручную без соблюдения временного регламента.
